Кнопка "Выбрать excel файл" - выбираем excel файл, который скачали с сервера через программу OpcTableCreator. Сортировать не обязательно, программа сама находит последний numTag, далее Файл1.
Кнопка "Выбрать KepToTable файл" - выбираем файл KepToTable, в котором добавлены только теги с той зоны, которая нам нужна, далее Файл2
Кнопка "Сравнить файлы" - сравнивает теги, если есть несовпадения выводит их в списке справа. То есть выводит все теги, которые есть в Файле2, но нет в Файле1. 
Можно выбрать только те теги, которые нужны и нажать "Сохранить", после чего выбрать место и имя нового файла.

Конечный файл можно дополнительно проверить "Проверить файл на дубликаты". Происходит проверка первых двух столбцов и NumTag. Если где-то есть повторяющиеся значения, выводит сообщение об этом и после проверки открывается файл "log.txt", где перечислинены все повторы.
